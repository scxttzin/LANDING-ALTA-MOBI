# Alta Mobi — Landing Page

Landing page de vendas do **Alta Mobi**, aplicativo de gestão inteligente de frota para frotistas que locam veículos para motoristas de aplicativo (Uber e 99).

## O que tem na página

- **Demos ao vivo** simulando o app real: navegação entre abas, cadastro de motorista com digitação animada e o ciclo completo de cobrança PIX (motorista paga o semanal por QR Code → pagamento cai em Recebimentos no painel do frotista, com baixa automática).
- **Mobi app** — apresentação do assistente virtual que tira dúvidas e ajuda na configuração da conta, com suporte via chat e WhatsApp.
- **Planos**: Mensal R$ 32/mês · Anual R$ 200/ano (cards animados).
- Prova social: método validado por mais de 19 empresas parceiras, incluindo a [Flex Drive](https://scxttzin.github.io/APP-FLEX-DRIVE/).

## Estrutura

- [`index.html`](index.html) — a landing completa, 100% autocontida (HTML + CSS + JS em um único arquivo, sem dependências externas).

## Como rodar

É um arquivo estático — basta abrir o `index.html` no navegador ou servir a pasta com qualquer servidor HTTP.
