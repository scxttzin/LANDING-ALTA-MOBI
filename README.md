# Novu — Landing Page

Landing page de vendas do **Novu**, aplicativo de gestão inteligente de frota para frotistas que locam veículos para motoristas de aplicativo (Uber e 99).

## O que tem na página

- **Demos ao vivo** simulando o app real: navegação entre abas (sidebar de ícones, gráficos do dashboard, cards de veículos), cadastro de motorista com digitação animada, ciclo completo de cobrança PIX (motorista paga o semanal por QR Code → cai em Recebimentos no painel do frotista) e **manutenções automáticas** (motorista solicita → empresa agenda com oficina parceira → motorista confirma → histórico atualizado).
- **Novu app** — assistente virtual para o frotista e para os motoristas, com suporte via chat e WhatsApp.
- **Plano**: R$ 25/mês até 60 veículos · R$ 50/mês acima de 60.
- Prova social: método validado por mais de 19 empresas parceiras, incluindo a [Flex Drive](https://scxttzin.github.io/APP-FLEX-DRIVE/).

## Estrutura

- [`index.html`](index.html) — a landing completa, autocontida (HTML + CSS + JS em um único arquivo).
- [`assets/novu-icon.png`](assets/novu-icon.png) — ícone da marca.

## Como rodar

É um site estático — basta abrir o `index.html` no navegador ou servir a pasta com qualquer servidor HTTP.
